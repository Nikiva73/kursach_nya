Инструкция по запуску к приложению курсового проекта : "Регистрация на СТО"
24-КБ-ПИ2 Нуйя

Приложение для учета клиентов СТО с функцией аутентификации пользователей. Позволяет регистрировать клиентов,просматривать историю обслуживания после успешного входа в систему.

Технологии

Язык программирования: C#
Фреймворк/Платформа: .NET 12.0.
Тип приложения: С# Windows Forms

Настройка и запуск

Откройте файл .sln. нажмите f5 или запустите компилятор по кнопочке.
Убедитесь, что вы подключили NuGet пакет "Microsoft.VisualBasic"

Чтобы войти в систему и посмотреть записи вам нужно знать пароль администратора:
"admin123"
Чтобы добавить нового клиента заполните форму:имя,телефон,марка машины,выбор услуги. Нажать кнопку
